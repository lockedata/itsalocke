Fixing the Tiny Icons, Big Text issue on my XPS13

The Issue

I love my Dell XPS13. It’s fast, sleek and gorgeous. It does however have one little problem: the icon and text size. The text was always too big for the buttons and boxes and the icons were so small you could hardly see them. This made it hard to use my machine without an external screen (which doesn’t have that issue and should have been my first clue!)

So OK, really I should have figured this out sooner, but I just put up with it thinking it was a fact of life. For the most part it was just annoying, but then I started to run into problems that were a bit more serious, like the file browser not showing up in some instances!

So now it was personal, and I resolved to fix the issue once and for all. Here’s how I did it:

The Fix

I’m running Windows 10, so things may look different for you, but hopefully this is still valid.

The issue comes from the scaling being applied to combat the high-resolution, so you’ll need to go to your Display Settings (the simplest way is to right-click your desktop)

From there look for the scaling option and set it to 100%, this will make everything fit more naturally.

This works well but some things still look pretty squished, this is because the default resolution for this device is a whacky 3200 X 1800 which is well above what most programs are designed for. This is why the scaling is set as it is in the first place after all.

I would recommend changing the resolution to one more suited to most programs. Here I’ve gone for 1920 X 1080:

And now I finally have my file browser back!

If you want to quickly change the resolution back to the full for watching films you could look into this solution on Hey, Scripting Guy! or into a neat little program called Display Fusion.
